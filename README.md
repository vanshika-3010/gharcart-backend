# 🛠️ GharCart Backend – REST API

This is the backend server for the GharCart e-commerce application, built using Node.js, Express.js, and MongoDB. It provides RESTful APIs for user authentication, product management, cart handling, and order processing.

## 🌐 Live API

🔗 Base URL (Railway):https://gharcart-backend-production.up.railway.app/

🧰 Tech Stack
Node.js – Runtime environment

Express.js – Backend framework

MongoDB Atlas – Cloud database

Mongoose – ODM for MongoDB

JWT (JSON Web Tokens) – Authentication

bcrypt.js – Password hashing

dotenv – Environment variables

cors – Cross-origin requests
